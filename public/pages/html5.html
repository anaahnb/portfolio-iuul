Descrição

HTML5 é uma linguagem de marcação de tecnologia central da Internet usada para estruturar e apresentar conteúdo para a World Wide Web. A partir de outubro de 2014, esta é a quinta revisão final e completa do padrão HTML do World Wide Web Consortium (W3C). A versão anterior, HTML 4, foi padronizada em 1997.

Pontos positivos

  • Facilidade de aprendizado
  • Compatibilidade com todos os navegadores
  • Estruturação de código clara

Pontos negativos

  • Segurança
  • Dependência de outras tecnologias
  • Repetição de código
